Welcome to SAX-VSM homepage!

A Java library that implements SAX-VSM algorithm for interpretable time series classification.

(The website's content is inherited from JMotif project. Here are quick links to the source code and the paper).

Here, we discuss algorithms:

Time series discretization steps:

  • Z normalization of timeseries
  • PAA: Piecewise Aggregate Approximation of time series
  • SAX: Symbolic Aggregate Approximation of time series
  • Time series discretization via sliding window

Vector space model (VSM):

  • TF*IDF statistics
  • Cosine similarity

... and show in action:

SAX-VSM application for:

  • Time series classification
  • Time series characteristic pattern discovery
  • Hierarchical clustering
  • Spherical K-Means
